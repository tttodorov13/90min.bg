Add Facebook Login to WordPress site without any developer.

== Description ==

An easy-to-use WordPress Plugin to add facebook login to your site that allows your visitor to login with there facebook id and they can easily put comment and do anything in role criteria. This plugin is developed to meet the needs of administrators who would like to add facebook login without any developer.

Just put our shortcode [efl-login] and use it.


**Features**

- Easy shortcode plugin
- Small configuration needed
- Admin interface built on WordPress UI
- No Complicated database queries
- Extremely lightweight
- Thorough documentation
- Virtually no impact on site performance

== Installation ==

Install Easy Facebook Login just as you would any other WP Plugin:

1.  [Download Easy Facebook Login] from WordPress.org.

2.  Unzip the .zip file.

3.  Upload the Plugin folder (easy-facebook-login/) to the wp-content/plugins folder.

4. Go to [Plugins Admin Panel](http://codex.wordpress.org/Administration_Panels#Plugins "Plugins Admin Panel") and find the newly uploaded Plugin, "Easy Admin CSS" in the list.

5. Click Activate Plugin to activate it.

6. Click Settings to configure it.

[More help installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins "WordPress Codex: Installing Plugins")
